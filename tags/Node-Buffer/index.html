    Buffer的理解

    什么是Buffer

    Buffer是一个像Array的对象，但它主要用于操作字节。在引入 TypedArray 之前，JavaScript 语言没有用于读取或操作二进制数据流的机制。 Buffer 类是作为 Node.js API 的一部分引入的，用于在 TCP 流、文件系统操作、以及其他上下文中与八位字节流进行交互。
